# 物业后台管理系统-前端

### 后端代码仓库链接
![后端代码仓库](https://github.com/Pitayafruits/manage_backend)

### 运行项目
``` bash
# 安装项目依赖的库
npm install

# 开发者环境运行项目
npm run dev

# 打包项目
npm run build
```

### 软件架构
* 核心框架：Spring Boot 2.4.0
* 安全框架：JwtPermission 3.1.1
* 前端：Ant Design Vue 1.6.2
* 持久层框架：MyBatis-Plus 3.4.1
* 关系型数据库: Mysql 8.0.22
* 数据库连接池：Druid 1.2.3
* 缓存数据库: Redis 4.0.9
* 项目管理工具: Maven 3.3+
* 工具类：Hutool 5.5.1
