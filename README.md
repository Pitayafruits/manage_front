# 物业后台管理系统-前端

### 后端代码仓库链接
![后端代码仓库](https://github.com/Pitayafruits/manage_backend)

### 运行项目
``` bash
# 安装项目依赖的库
npm install

# 开发者环境运行项目
npm run dev

# 打包项目
npm run build
```

### 技术栈
* vue2
* vuex
* vue-router
* axios
* elementUI
* 阿里图标iconfont
